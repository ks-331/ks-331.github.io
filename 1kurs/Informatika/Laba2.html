Лабораторная работа: Создание исполняемых файлов

С помощью командной строки создать каталог lab2 в каталоге D:\Student

В каталоге lab2 создать текстовый файл, содержащий в себе справку по команде tree.

Создайте файл 1.txt с текстом «Hello » и файл 2.txt с текстом «World!». Затем выполните сведение файлов 1.txt и 2.txt в 3.txt

Откройте блокнот и создайте командный файл в папке D:\Student назовите его ByMe.bat, который должен выполнять следующие действия:

  1. создать переменную MyName присвоить ей ваше имя (желательно использовать только символы латинского алфавита, чтобы избежать несоответствия кодировки)
  2. создать переменную MySurname присвоить ей вашу фамилию
  3. вывести переменные на экран
  4. объединить переменные MyName и MySurname в переменную My
  5. создать переменную Group присвоить ей значение по названию вашей группы, но в начале поставить 6 символов *, например ******КС-131
  6. создать переменную My1 в которой нужно заменить символы ****** на строку group
  7. записать в файл My.txt переменные My и My1
  8. очистить переменную My
  9. отключить дублирование команл

Создайте еще один командный файл в этой же директории и назовите его auto.bat. Файл должен выполнять следующие действия:

  • вывести сообщение, предлагающее пользователю открыть программу Internet Explorer. Если пользователь нажмет сочетание клавиш ctrl+C, исполняемый файл завершит свою работу, если будет нажата иная клавиша файл запустит программу. Файл находится C:\Program Files\Internet Explorer\iexplore.exe

Сохраните изменения.

Откройте с помощью блокнота файл ByMy.bat, который мы создали первым и внесите в него изменения так, чтобы он запускал auto.bat
